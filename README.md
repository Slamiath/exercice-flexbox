Une galerie d'images moderne, fluide et entièrement responsive, conçue avec HTML5 et CSS3. Ce projet met l'accent sur l'expérience utilisateur (UX) avec des animations de survol et une structure adaptative (Flexbox).

# Fonctionnalités
-Design Responsive : S'adapte parfaitement aux ordinateurs, tablettes et mobiles grâce aux media queries.
-Interactivité Fluide : Effets de zoom (scale) au survol des cartes pour une navigation dynamique.
-Optimisation de l'Affichage : Gestion intelligente des débordements (overflow) pour éviter les barres de défilement inutiles.

# Technologies utilisées
HTML5 : Structure sémantique du contenu.
CSS3 :
-Flexbox pour l'alignement et la mise en page.
-Transitions & Transforms pour les animations.
-Media Queries pour la compatibilité mobile.
